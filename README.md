# Weather Dashboard 🌤️

A dynamic and responsive Weather Dashboard application that provides real-time weather data and forecasts. Users can search for any city globally to get current weather conditions, including temperature, humidity, and wind speed.

## 🚀 Features

* **Real-time Data:** Fetches live weather information using a third-party weather API.
* **City Search:** Intuitive search functionality to find weather details for specific locations.
* **Detailed Metrics:** Displays temperature, humidity, wind speed, and weather descriptions (e.g., cloudy, sunny, rainy).
* **Responsive UI:** Optimized for both desktop and mobile viewing.

## 🛠️ Tech Stack

* **Frontend:** HTML5, CSS3, JavaScript (ES6+) / React *(удали лишнее)*
* **Backend:** Python (Flask) / Node.js *(удали лишнее, если приложение работает только на клиенте)*
* **API:** OpenWeatherMap API / WeatherAPI *(впиши тот, который использовал)*
